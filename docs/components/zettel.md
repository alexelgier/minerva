# Zettel Agent - Detailed Documentation

See [Architecture Documentation](../architecture/zettel.md) for technical details.

This document provides comprehensive information about the zettel agent system.

## Overview

zettel is a LangGraph-based agent system for processing book quotes and extracting atomic concepts (Zettels) using Zettelkasten methodology.

## Key Features

- Quote parsing from markdown files
- Concept extraction using vector search
- Quote attribution to existing concepts
- Neo4j integration with vector indexes
- Batch processing capabilities
