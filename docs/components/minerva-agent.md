# Minerva Agent - Detailed Documentation

See [Architecture Documentation](../architecture/minerva-agent.md) for technical details.

This document provides comprehensive information about the minerva_agent LangGraph agent.

## Overview

minerva_agent is a deep agent built with LangGraph and deepagents that provides intelligent assistance for Obsidian vault management.

## Key Features

- File system operations (read, write, edit)
- Search and discovery (glob patterns, grep)
- Task planning and execution
- Subagent delegation
- Bilingual support (Spanish/English)
